## smoke and fire object detection
dataset: https://universe.roboflow.com/latifa-sassi-zqgnz/fire-smoke-mx4z8
computer vision model architecture used: https://yolov8.com/

## libraries used
* ultralytics
* roboflow
* os
* PIL
* IPython

## training
all images in the dataset were used with a batch size of 32 over 32 epochs.

# metrics
* precision: 0.9897617643329505
* recall: 0.9916134512944791
* mAP50: 0.9938636764904991
* mAP50-95: 0.9800504252151985
* fitness: 0.9800504252151985
